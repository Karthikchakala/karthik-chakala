### 👤 About Me

I am a driven Full Stack and Backend Developer with over 2 years of hands-on project experience building scalable web applications and integrating AI technologies. I love solving complex structural problems, optimizing databases, and bridging the gap between clean backend architecture and intuitive frontend interfaces. 

Beyond coding, I actively share my journey and tech insights across social platforms like Instagram, YouTube, and Medium to give back to the developer community!

* 🚀 **Current Focus:** Advanced AI orchestration using LangChain, RAG pipelines, and WebRTC systems.
* 🎓 **Status:** Software Engineering Student constantly exploring new paradigms.
* 💡 **Philosophy:** Write code that is clean, readable, and highly maintainable.
* 🎧 **Fun Fact:** I love building real-time apps because watching immediate data sync is incredibly satisfying.
